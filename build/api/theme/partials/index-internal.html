JindoJS는 nhn에서 제작한 javascript library입니다.

JindoJS Component는 JindoJS 를 기반으로 개발된 재사용 가능한 독립적인 자바스크립트 모듈입니다. JindoJS는 Javascript Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화 하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다.

사용방법

우상단 전체 다운로드 또는 선택 다운로드 링크를 통해 JS 파일을 다운로드 받아 HTML 파일에 아래와 같이 삽입하여 사용합니다. 
<script type="text/javascript" src="jindo_component.js"></script>
만약 다른 이름으로 변경된 Jindo 와 함께 사용한다면 선택 다운로드의 '상세옵션' 에서 변경 된 이름을 지정하여 다운로드 받거나, 아래와 같이 파라미터를 지정하여 변경된 Jindo 의 이름을 지정 할 수 있습니다. 
<script type="text/javascript" src="jindo_component.js?jindo=FOO"></script>
